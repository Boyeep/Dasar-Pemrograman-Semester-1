Program ini dirancang untuk menghitung luas, keliling, dan titik pusat lingkaran (circumcenter) dari sebuah segitiga berdasarkan tiga titik koordinat yang dimasukkan oleh pengguna. Sesuai ketentuan, setiap fungsi dipisah ke dalam header file terpisah agar kode lebih modular. Fungsi luas_segitiga() pada area.h menggunakan rumus shoelace untuk menghitung luas segitiga sembarang. Pada perimeter.h, fungsi jarak() digunakan untuk menghitung panjang sisi dengan rumus jarak Euclidean, lalu dijumlahkan dalam keliling_segitiga() untuk mendapatkan keliling segitiga. Sementara itu, fungsi circumcenter() pada circumcenter.h menghitung titik pusat lingkaran yang melalui ketiga sudut segitiga dengan menggunakan rumus matematika berbasis determinan. Di file main.c, program membaca input koordinat titik dari pengguna, memanggil ketiga fungsi tersebut, lalu menampilkan hasil perhitungan. Jika titik-titik segaris sehingga tidak dapat membentuk segitiga, program menampilkan pesan bahwa titik lingkaran tidak valid. Dengan pemisahan fungsi melalui header, program menjadi lebih terstruktur, mudah dipelihara, dan sesuai konsep modular programming.
