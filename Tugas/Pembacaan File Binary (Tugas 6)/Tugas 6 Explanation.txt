Program ini digunakan untuk membaca seluruh isi file biner berisi data penduduk berdasarkan struktur penduduk. File dibuka dalam mode biner menggunakan fopen("WargaKebumen.dat", "rb"). Setiap record dibaca satu per satu dengan fread, lalu ditampilkan dalam format kolom yang rapi. Sebelum dicetak, program mengecek jika nilai tanggal_lahir lebih dari 40, maka dikurangi 40 terlebih dahulu karena itu menandakan gender perempuan pada data. Usia dihitung menggunakan data tahun lahir, dan hanya data penduduk dengan usia balita (0–5 tahun) serta siap produktif (18–60 tahun) yang ditampilkan. Format output mengikuti aturan: nama lebar 40 karakter rata kiri, tanggal lahir ditulis dalam format dd-mm-yy, dilanjutkan desa dan kecamatan masing-masing lebar 20 karakter rata kiri. Program juga menangani kemungkinan data kosong atau rusak, dan akan berhenti membaca ketika mencapai akhir file. Dengan format printf yang tepat menggunakan padding (%-40s, %02d-%02d-%02d, dll.), kolom output dijamin lurus dari atas ke bawah sesuai ketentuan soal.
